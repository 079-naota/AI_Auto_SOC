# 新しいサーバーとその監視対象の提案

## 新サーバー3台の場合の監視対象

### 1.Webアプリケーションの監視
Webハニーポットで、悪意のあるURLや何読かされたリクエストを捕獲する。

まず、Snare＆Tannerをインストールし、`80/443port`で起動する。
 15 次にWazuhAgentをインストールする。
 16 最後に、Snare等の出力ログファイルを読んでManagerへ送信する。


必要スペック
- CPU:1 コア
- RAM:1~2GB
- SSD:20GB


### 2.データベースの監視
攻撃者はデータを盗むだけでなくデータを破壊し、身代金を要求する場合がある。
その際のツールやIoCを捕獲する。

まず、RedisHoneypotをインストールし、`6379port`で起動する。
次に、WazuhAgentをインストールする。
最後に、RedisHoneypotが出力する脅迫文やコマンドログをWazuhAgentに読ませ、Managerへ送信する。


必要スペック
- CPU:1コア
- RAM:512MB~1GB
- ストレージ:20GB


### 3.マルウェア収集
Windows特有の脆弱性やルータ・カメラなどのIoT機器を狙うボットネットの活動を監視する。

まず、Dionaeaをインストールし、`445/21port`で起動する。
次に、WazuhAgentをインストールする。
最後に、攻撃のテキストログはWazuhAgentで転送する。

必要スペック
- CPU:1~2コア
- RAM:1~2GB
- SSD:30~50GB
